Instrukcja Api

Do korzytania z Api wymagany jest klucz

Żeby dostac bezpłatny klucz API proszę wysłać maila na adres stan.robert.1986@gmail.com

Lista dostępnych walut na których można wykonać przeliczenia

{freeApiKey} - darmowy klucz API

http://localhost:8080/api/currencies/codes?apiKey={freeApiKey}

List aktualnych kursów dla listy walut

http://localhost:8080/api/currencies?apiKey={freeApiKey}

Przeliczenie na podstawie kursu wymiany dla następujących parametrów: kwota, z jakiej waluty, do jakiej waluty

{currency_from} - z jakiej waluty chcesz wymienić

{currency_to} - na jaką walutę chcesz wymienić

{number} - kwota

http://localhost:8080/api/currencies/convert?from={currency_from}&to={currency_to}&amount={number }&apiKey={freeApiKey}

Przykładowe zapytanie

http://localhost:8080/api/currencies/convert?from=EUR&to=PLN&amount=100&apiKey={freeApiKey}
